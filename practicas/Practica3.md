# Conceptos:
* Planificación
  * Seleccionar al proceso a ejecutar.
  
* Cambio de Contexto:
  * Cambio que ocurrre cuando el cpu deja de ejecutar un proceso y ejecuta otro.
  * Guardar el contexto stack pointer. Program counter
  
* Estados de un proceso
